# Spring Boot 2 REST API

## Create project using maven
```
mvn archetype:generate -DgroupId=com.rest.api -DartifactId=rest-api -Dversion=1.0 -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false
```
